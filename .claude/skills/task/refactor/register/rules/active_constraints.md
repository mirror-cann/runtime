# 全局共享约束

## 注册机制迁移通用约束
- **目标架构**：任务注册机制从"机制层集中赋值"重构为"任务文件静态注册"
- **注册函数放置**：在每个任务版本的实现文件末尾新增 `{TaskName}TaskRegister` 静态注册函数
- **命名规范**：注册函数名 `{TaskName}TaskRegister`，静态变量名 `g_{taskName}TaskRegister`
- **目录结构**：任务文件保持现有版本目录结构（v100/、v200_base/、v200/、v201/），注册函数追加到对应版本文件末尾，不新增独立目录
